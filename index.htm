About

Milwaukee Azure is a user group of IT people from the Milwaukee, WI area that have gotten together in a common interest surrounding the Microsoft Azure cloud platform. The group was started in May 2016 when Chris Pietschmann started the Milwaukee Azure Slack Group in an effort to get the conversations started.

The Milwaukee Azure website hosted at MkeAzure.com is hosted as an Open Source project on Github utilizing Github Pages. This not only provides a free way to host the website, but also an extremely convenient and open method of the Milwaukee Azure community to manage the website as a whole.
